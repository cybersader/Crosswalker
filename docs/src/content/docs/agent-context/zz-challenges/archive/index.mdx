## What's here

Challenges that started as fresh-agent research briefs and have been **resolved** — meaning a deliverable was produced and Crosswalker committed to a direction in a dated log. The original brief is **preserved verbatim** so the same challenge can be re-run later (e.g., when external conditions change) without losing the original framing.

Each archived challenge has a top-of-file callout pointing to the resolving log entry.

## When to re-run an archived challenge

A challenge is archived as resolved, not closed. Re-run when:

- An assumption the resolution depends on changes (the resolution callout typically lists these)
- New research surfaces that contradicts the committed direction
- A fresh agent wants an independent perspective on a load-bearing decision

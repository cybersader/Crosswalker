## Available commands

Access these via the command palette (`Ctrl/Cmd + P`):

| Command | Description |
|---------|-------------|
| **Crosswalker: Import structured data** | Opens the 4-step import wizard |

## Planned commands

These are on the roadmap for future releases:

| Command | Description |
|---------|-------------|
| Insert framework link | Search and insert a typed link to a framework node |
| Browse saved configs | Open the config browser modal |
| Re-import framework | Re-run import with an existing config |

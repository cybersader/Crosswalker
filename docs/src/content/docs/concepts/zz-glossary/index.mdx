This is the **verbose glossary** — one page per term when a table row isn't enough. For quick-reference definitions, see the [terminology](/crosswalker/concepts/terminology/) page.

Terms are added here when they need more than a one-liner — examples, diagrams, history, or connections to multiple concepts.

### Adding a term

Create `YYYY-MM-DD-term-name.mdx` or just `term-name.mdx` in this folder:

```yaml
---
title: Term name
description: One-sentence definition.
tags: [relevant-tags]
sidebar:
  label: "Term name"
---
```

If the term has temporal context (discovered during a session), use the date prefix. If it's a permanent reference term, skip the date.

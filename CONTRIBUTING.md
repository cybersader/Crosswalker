# Contributing to frameworks_to_obsidian

:rocket: We welcome contributions of all kinds! Please take a moment to review these guidelines.

## How to Contribute
1. Fork the repository and create a new branch:
   ```bash
   git checkout -b feature/your-feature-name
   ```
2. Install dependencies and set up your environment (see `README.md`).
3. Make your changes
4. Add or update tests if applicable.
5. Commit your changes with clear messages:
   ```
   feat: add support for new framework
   fix: correct tag aggregation bug
   doc: update README for installation
   ```
6. Push your branch and open a Pull Request.

## Reporting Issues
- Use GitHub Issues to report bugs or request new features.
- Provide a clear title and description, including steps to reproduce.

# Crosswalker Roadmap

## Current (0.1.x)
- Bug fixes and UI polish
- Improve error handling and user feedback
- Documentation site launch

## Near-term
- **XLSX parser** — Excel file support (xlsx package installed, needs integration)
- **JSON parser** — Import from JSON/JSONL files
- **Config export/import** — Share import configurations as JSON files
- **Improved preview** — Expandable folder tree, full note preview

## Medium-term
- **Cross-framework linking** — Generate typed links between imported frameworks
- **Link insertion commands** — "Insert framework link" with search modal
- **Autocomplete** — Suggestions for framework references in notes
- **Batch operations** — Re-import/update existing framework folders

## Long-term
- **OSCAL export** — Export to Open Security Controls Assessment Language
- **Community templates** — Shareable configs for common frameworks (NIST, CIS, MITRE)
- **Framework update/sync** — Detect and apply upstream framework changes
- **Report generation** — Compliance dashboards and gap analysis

## Infrastructure
- [ ] E2E testing in CI with wdio-obsidian-service
- [ ] Community plugin submission to Obsidian plugin registry
- [ ] Automated changelog generation
- [ ] Plugin usage analytics (opt-in)
